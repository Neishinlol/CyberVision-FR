## 📖 À propos
CyberVision est une **énorme modlist pour Cyberpunk 2077** avec plus de **1500 mods**, conçue pour un gameplay immersif et une fidélité visuelle maximale.  
Axée sur le **roleplay, le réalisme et la qualité de vie**, elle supporte les **profils anglais et français**.  
Fonctionne au mieux en **difficulté Very Hard**.

Pour les **instructions détaillées de configuration des mods, le setup complet et les réponses à vos questions**, consultez la **section CyberVision du Discord** : [Lien Discord](https://discord.com/invite/WSFgf5S7wS).

## ⚙️ Pré-Installation
Avant d’installer CyberVision, complétez les étapes suivantes :

### **Utilitaires requis**
- **.NET SDK** :  
  - [.NET 8.0 SDK - Windows x64 Installer](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)  
  - [.NET 9.0 SDK - Windows x64 Installer](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)
- **Microsoft Visual C++ Redistributable** :  
  - [x86](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)  
  - [x64](https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170)

### **Contenu officiel du jeu**
- Assurez-vous que votre jeu est propre et à jour  
- Lancez-le une fois pour initialiser REDmod (les utilisateurs Steam doivent télécharger REDmod pour Cyberpunk 2077 avant)

### **Steam Overlay**
- Désactivez le Steam overlay globalement ou uniquement pour Cyberpunk afin d’éviter les crashs :  
  1. Steam → Paramètres → Dans le jeu → Décochez “Activer le Steam Overlay”  
  2. OU Bibliothèque → Cyberpunk 2077 → Clic droit → Propriétés → Décochez “Activer le Steam Overlay”  

### **Emplacement d’installation**
- **Ne pas installer dans C:\Program Files** ou C:\Program Files (x86)  
- Recommandé : `C:\Games\Cyberpunk` ou `D:\Modlist\CyberVision`  

### **Antivirus**
- Désactivez ou supprimez les antivirus tiers (Webroot, Bitdefender)  
- Ajoutez le **dossier Wabbajack list** à l’exclusion de l’antivirus  

### **Pagefile / Mémoire virtuelle**
Les grosses modlists nécessitent beaucoup de mémoire. Quelle que soit votre RAM/VRAM, cette étape est **obligatoire** :  
1. Win + R → `sysdm.cpl ,3` → Performance → Avancé → Mémoire virtuelle → Modifier  
2. Décochez "Gérer automatiquement"  
3. Sélectionnez le SSD le plus rapide → Taille personnalisée : `40960 MB` initial et max → Définir → Appliquer → Redémarrez  

### **Utilisateurs NVIDIA - Cache de shaders**
- Panneau de configuration NVIDIA → Gérer les paramètres 3D → Taille du cache de shaders → Mettre à **10 GB** → Appliquer

## 🛠 Installation
1. Supprimez toutes les données Cyberpunk  
2. Lancez une fois pour activer REDmod  
3. Installez CyberVision via Wabbajack  
4. Sélectionnez le profil MO2 (12 profils : 6 anglais, 6 français)  

Pour **les instructions de setup, la configuration détaillée et le support**, consultez la **section CyberVision du Discord** : [Lien Discord](https://discord.com/invite/WSFgf5S7wS)
